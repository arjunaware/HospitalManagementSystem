# HospitalManagementSystem
Hospital Management System is a Spring Boot backend application that provides RESTful APIs for managing doctors, patients, and appointments. It includes CRUD operations, pagination, exception handling, validations, logging with SLF4J/Log4j2, and monitoring using Spring Boot Actuator, following a clean layered architecture with Spring Data JPA.




Hospital Management System is a backend application built using **Spring Boot**.  
This project provides **RESTful APIs** to manage doctors, patients, and appointments and demonstrates real-world backend development concepts.

It is suitable for **Java backend learners**, **interview preparation**, and **portfolio showcasing**.
